# RedSquare Screens - Amazon Fire TV Installation Guide

## Installation Instructions

### For Amazon Fire TV/Fire Stick Owners:

1. **Enable Apps from Unknown Sources:**
   - Go to Settings → My Fire TV → Developer Options
   - Turn ON "Apps from Unknown Sources"
   - Turn ON "ADB Debugging" (optional, for advanced users)

2. **Install RedSquare Screens:**
   - Download the latest `.apk` file from the releases to a USB drive
   - Connect USB drive to your Fire TV (or use a file manager app)
   - Navigate to the APK file and install
   - Allow installation when prompted

### Alternative Installation Methods:

**Method 2 - Downloader App:**
   - Install "Downloader" from Amazon App Store
   - Open Downloader and enter the APK download URL
   - Install directly on your Fire TV

**Method 3 - ADB (Advanced Users):**
   - Enable ADB debugging on Fire TV
   - Connect via ADB: `adb connect [Fire TV IP]`
   - Install APK: `adb install screens-amazon-fire-app.apk`

3. **Setup Your Screen:**
   - Launch RedSquare Screens from your Fire TV apps
   - Note the Screen ID displayed prominently
   - Visit redsquare.app on any device to register your screen
   - Configure availability times, pricing, and content preferences

## Remote Control Guide:

- **Home Button:** Return to Fire TV home (app remains running in background)
- **Back Button:** Access screen settings menu
- **Select/OK Button:** Interact with on-screen elements
- **Menu Button:** Quick access to screen controls
- **Voice Button:** "Alexa, open RedSquare Screens"

## Performance Tips:

- Close other apps to maximize performance
- Ensure stable internet connection (5+ Mbps recommended)
- Use Ethernet connection for best results
- Restart Fire TV weekly for optimal performance

## Support & Troubleshooting:

**Installation Issues:**
- Ensure "Apps from Unknown Sources" is enabled
- Check available storage space (minimum 100MB required)
- Restart Fire TV and try again

**App Not Launching:**
- Clear cache: Settings → Applications → Manage Installed Applications → RedSquare Screens → Clear Cache
- Reinstall the app if issues persist

**Content Not Displaying:**
- Verify internet connection
- Check screen registration at redsquare.app
- Ensure no content blocking settings are active
