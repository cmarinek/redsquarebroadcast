# RedSquare Screens - Samsung TV Installation Guide

## Installation Instructions

### For Samsung Smart TV Owners (2017+):

1. **Enable Developer Mode:**
   - Press Info → Menu → Mute → 1 → 1 → 9 → Enter on your Samsung remote
   - Turn ON "Developer mode" 
   - Enter PC IP address for development server
   - Restart your TV

2. **Install RedSquare Screens:**
   - Download the latest `.tpk` file from the releases
   - Use Samsung's Smart TV SDK or sideload via developer tools
   - Install the TPK package on your TV

### Alternative Installation (Samsung App Store - Coming Soon):
   - Search for "RedSquare Screens" in the Samsung Smart TV App Store
   - Install directly from the store
   - No developer mode required

3. **Setup Your Screen:**
   - Launch RedSquare Screens from your TV's app menu
   - Note the Screen ID displayed on screen
   - Visit redsquare.app to register your screen
   - Configure availability, pricing, and content preferences

## Remote Control Guide:

- **Menu Button:** Access screen settings
- **Back Button:** Return to main display
- **Color Buttons:** Quick actions (Red: Emergency stop, Green: Connection test)
- **Arrow Keys:** Navigate menus when active

## Troubleshooting:

**App won't install:** Ensure developer mode is enabled and your TV firmware is up to date
**No content displaying:** Check internet connection and verify screen registration
**Remote not responding:** Press and hold the power button for 10 seconds to restart
