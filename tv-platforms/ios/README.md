# RedSquare Screens - iOS Platform

Transform your iPhone or iPad into a mobile advertising display or casting controller with RedSquare Screens.

## Overview

RedSquare Screens for iOS enables your iPhone or iPad to serve as both a content management tool and a casting device for digital advertising displays, allowing you to control and monitor nearby screens.

## Configuration

### Initial Setup
1. Download and install RedSquare Screens iOS app
2. Sign in with RedSquare account or create new account
3. Grant necessary permissions (Camera, Photos, Local Network)
4. Choose app mode: Display, Controller, or Casting
5. Connect to WiFi network (required for cloud features)

### Display Setup (iPad Mode)
1. Select "Display Mode" in app settings
2. Register device as display screen with unique ID
3. Configure display schedule and content preferences
4. Set up charging station for continuous operation
5. Enable guided access mode to prevent user interference

### Controller Setup
1. Select "Controller Mode" for device management
2. Allow location services for nearby display discovery
3. Connect to displays via QR code scan or manual entry
4. Configure notification preferences for display alerts
5. Set up content upload preferences and quality settings

## Troubleshooting

### Common Issues

#### App Crashes or Freezes
- **Cause**: Insufficient memory or iOS version compatibility
- **Solution**: Restart app, update iOS, free up storage space

#### Content Not Uploading
- **Cause**: Network connectivity or file format issues
- **Solution**: Check WiFi connection, verify supported file formats

#### Display Not Connecting
- **Cause**: Network connectivity or firewall restrictions
- **Solution**: Verify same network, check firewall settings

#### Poor Video Quality
- **Cause**: Network bandwidth or device performance limitations
- **Solution**: Reduce video quality, use WiFi instead of cellular

### Performance Issues

#### Slow Content Loading
- Check internet connection speed and stability
- Clear app cache and restart application
- Reduce content file sizes for faster loading
- Use WiFi instead of cellular data when possible

#### Battery Drain in Display Mode
- Enable Low Power Mode in app settings
- Reduce screen brightness manually
- Close unnecessary background apps
- Use iPad with continuous charging setup

### Network Issues

#### Cannot Discover Nearby Displays
- Ensure both devices are on same WiFi network
- Check local network permissions in iOS Settings
- Verify display devices are powered on and connected
- Try manual connection using display IP address
