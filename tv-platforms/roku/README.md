# RedSquare Screens - Roku Installation Guide

## Installation Instructions

### For Roku Device Owners:

1. **Enable Developer Mode on your Roku:**
   - Press Home 3 times, Up 2 times, Right 1 time, Left 1 time, Right 1 time, Left 1 time, Right 1 time
   - Set a developer password when prompted
   - Restart your Roku device

2. **Install RedSquare Screens:**
   - Download the latest `.zip` file from the releases
   - Open a web browser and navigate to your Roku's IP address
   - Login with your developer credentials
   - Upload the RedSquare Screens channel package
   - Install and launch the channel

3. **Setup Your Screen:**
   - Launch RedSquare Screens from your Roku home screen
   - Note the Screen ID displayed on screen
   - Visit redsquare.app to register your screen
   - Set your availability and pricing
